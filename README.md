El resultat ha de ser pixel perfect (màxima fidelitat al disseny: mides, espaiats, tipografies, vores, alineacions i colors).

Condicions
Només Tailwind (CDN o build). Sense Bootstrap ni CSS personalitzat (excepte l’imprescindible per carregar una font si s’indica).
No es requereix funcionalitat (dropdowns, sliders, fletxes… poden ser només visuals).
Maquetació responsive obligatòria.
Seccions que has de construir (en aquest ordre)
Header / Navbar — logo + enllaços + idioma + botó login
Hero principal — text + CTA + imatge destacada
Bloc cercador + tags — input + chips/hashtags
Upcoming concerts — títol + “See All” + grid de cards
Popular near you — títol + “See All” + grid de cards
Explore by category — categories en cards
Artists — fila/carrossel visual d’avatars amb fletxes
The latest in music, tours & artists — secció fosca amb cards tipus blog
What our customers say — testimoniatges en cards
Footer — newsletter + columnes d’enllaços + icones + botigues
Criteris de correcció
Cada secció valdrà 1 punt. Es valorarà el nivell de similitud entre el prototip i el resultat.
